    Input Value : hashmap.put(key, value)
    Get Value : hashmap.get(key)
    Get Value or default : hashmap.getOrDefault(key, DefaultValue)

    Check presence of a key in hashmap : hashmap.containsKey(key)
    Create a list of keys in hashmap : hashmap.keySet()
    Create a set of entries in hashmap : hashmap.entrySet()
